Counter Strike:Global Offensive
Counter-Strike: Global Offensive expands upon the team-based action gameplay that it pioneered with Half-Life: Counter-Strike. CS: GO features new maps, characters, and weapons, and delivers updated versions of the classic CS content. In addition, CS: GO introduces new gameplay modes, matchmaking, leader boards, and more.Global Offensive, like previous installments, operates in short rounds that end when all players on one side are dead or an objective is completed.

Global Offensive is targeted for release in early 2012. It will be available for Microsoft Windows, Mac OS X, Xbox 360 via the Xbox Live Arcade and PlayStation 3 via PlayStation Network. It will feature classic content, such as old maps, as well as brand new maps, characters and game modes.
